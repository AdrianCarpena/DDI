El proyecto es un formulario con validaciones en tiempo real, la inserción de presonalidades mediante un selector y 2 botones de agregar y quitar, un checkbox para aceptar las condiciones...
Cuando todos los campos están correctamente puestos y estan aceptadas las condiciones, al dar a enviar saldrá un resumen de los datos introducidos; en caso de haber algún campo mal puesto nos 
indicará cual para que lo cambiemos. También dispone de un botón para limpiar el formulario y empezar de 0.
Hemos intentado mediante el css darle un aspecto "Divino" para que parezca un formulario para un juego de rol donde estás creando el perfil, poniendo datos del usuario y de su rol o personaje.
